NERT FORUM
The NERT Forum is a place to encourage involvement and community among graduates of the NERT Program.
BACKGROUND

NERT Forum was born to help solve the problem of volunteer engagement and keep the graduates more involved in their community. I built the forum in Ruby on Rails to highlight user generated Events and Posts, utilizing Google APIs and Ruby Gems to create an inviting user experience. I also integrated Devise, CanCan and my own methods to limit user access and functionality by administrative type.

CITY HYKE
City Hyke is a collection of user-generated city tours, complete with walking direction routes.
BACKGROUND

This MEN stack project uses Google APIs to dynamically draw tour routes based on user added stops. I worked in a two-person team to create this application, employing pair programming techniques to meet a small-sprint deadline. Dynamic CRUD actions were implemented by using AJAX for seamless update and delete actions. Page templating was implemented by employing EJS.
TECHNOLOGIES
  • MongoDB
  • Mongoose
  • Express Node
  • AJAX
  • EJS
  • Google Maps/Places API
